# Дашборд образовательных метрик для Grist

Готовый read-only Custom Widget для таблицы с данными по потокам. Виджет показывает:

- фильтры по курсу, году и кварталу;
- переключение детализации «квартал / месяц»;
- KPI: число студентов, окончившие, CoR и успеваемость;
- динамику процентных показателей;
- количество студентов по периодам;
- сравнение курсов;
- детальную таблицу потоков;
- предупреждения о строках без года/квартала, потоках без окончивших и некорректной базе для расчёта успеваемости.

## Расчёт показателей

- `CoR = сумма окончивших / сумма всех студентов`.
- `Успеваемость = сумма набравших >50% / сумма окончивших` по строкам, где число набравших не превышает число окончивших.

Проценты считаются из числовых колонок, поэтому итог является взвешенным и не зависит от размера отдельных потоков.

## Публикация через GitHub Pages

1. Создайте новый публичный или приватный репозиторий GitHub (для приватного проверьте доступность Pages на вашем тарифе).
2. Загрузите в корень репозитория файлы `index.html`, `styles.css`, `app.js`.
3. В GitHub откройте **Settings → Pages**.
4. В разделе **Build and deployment** выберите **Deploy from a branch**, ветку `main`, папку `/ (root)`.
5. После публикации скопируйте адрес страницы вида `https://USERNAME.github.io/REPOSITORY/`.

Для предварительного просмотра вне Grist добавьте к адресу `?demo=1`.

## Подключение в Grist

1. На странице документа выберите **Add New → Add Widget to Page**.
2. Выберите **Custom**, затем таблицу с данными.
3. В настройках виджета укажите опубликованный URL.
4. Разрешите уровень доступа **Read selected table**.
5. Сопоставьте поля:

| Поле виджета | Колонка Grist |
|---|---|
| Курс | Курс |
| Поток | Поток |
| Месяц окончания | Месяц окончания |
| Квартал окончания | Квартал окончания |
| Год окончания | Год окончания |
| Всего студентов | Всего студентов |
| Окончили обучение | Окончили обучение |
| Набрали >50% | Набрали >50% |

Колонки `CoR` и `Успеваемость` сопоставлять не нужно: виджет рассчитывает их из количественных полей.

## Требования к данным

- Месяц: число от 1 до 12.
- Квартал: число от 1 до 4.
- Год: четырёхзначное число.
- Количественные показатели: числовые колонки.
- Строки без года или квартала остаются в детальной таблице, но не попадают на временные графики.
